Using mission packs by moving them to the "id1" folder

Please follow this steps to use mission packs or mods by moving them to the "id1" folder:

1 Quit Quake.
2 Rename the file "PAK0.PAK" within the mission pack folder to "PAK2.PAK".
3 Move the file "PAK2.PAK" from the folder of the mission pack to the "id1" folder.
4 Start Quake.
