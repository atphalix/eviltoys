Quake does not respond anymore to key or mouse input if I insert an AudioCD

This problem does only occur with MacOS X v10.1 or later: If you insert a new Audio-CD while the game is already running, iTunes will be started automatically in the background. iTunes will get the active application but will not be visible if you are in fullscreen mode. To gain back control over Quake you should hit the mouse button once or hit "Command" and "Tab" until controls are back.

Under MacOS X v10.1.x do the following to suppress the autostart of iTunes:

1 Download and install the free utility "TinkerTool".
2 Open "System Preferences".
3 Switch to the "TinkerTool" preferences panel.
4 Disable the autostart of iTunes at the "TinkerTool" preferences panel.

Or you can do the following if you don't want to install the "TinkerTool" preferences panel:

1 Open "Terminal" which comes with the MacOS X installation.
2 At the Terminal prompt enter "defaults write com.apple.finder AutoplayAudioCD 0" and hit return.
3 Log out and in so that the new defaults value can take effect.

Under MacOS X v10.2 or later do the following to suppress the autostart of iTunes:

1 Open "System Preferences".
2 Switch to the "CDs & DVDs" System Preferences panel.
3 Set "When you insert a music CD" to "ignore".
